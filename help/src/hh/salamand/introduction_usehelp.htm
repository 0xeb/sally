Sally: Introduction

Using Help

This is context sensitive help system, so you can ask for help directly from opened dialog boxes, on chosen menu items, or on chosen buttons in toolbars, see Accessing Help for details.

The manual is separated into several major sections:

Window Components

This section describes Sally windows and controls.

Using Sally

This section describes how to use Sally for managing files and directories. Particular tasks are separated according to user experience to basic and advanced tasks.

Dialog Boxes

This section describes all dialog boxes and can be quickly referred to when you have a specific question about some dialog option.

Shortcuts

This section contains overview of keyboard and mouse shortcuts usable in Sally.

Plugins

This section describes plugins for Sally. Each plugin has its own help where you can find details on purpose, using, configuring, and keyboard shortcuts.

Appendix

This section covers a variety of topics and can be used for general information, as well as reference.

Glossary

This section contains definitions for terms used in the documentation of Sally.
