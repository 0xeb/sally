Sally: Plugins: Check Version: Dialog Boxes

Check For New Versions

Use this window to check for newer stable releases of Sally. It connects to GitHub Releases and gets information about the latest published version.

Options

Log window
It contains progress of checking for newer releases after clicking on Check Now. When all necessary information is obtained, it shows the current installed version, the latest GitHub release, a direct download link for the matching platform when available, and a link to the GitHub release page. Right click a link to copy its URL to the clipboard.
Configure
Use this button to open the Configuration of Check Version dialog box.
Check Now
It starts checking for the latest stable Sally release on GitHub.
