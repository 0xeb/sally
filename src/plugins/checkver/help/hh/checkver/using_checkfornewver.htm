Checking For New Versions

Use the Check Version plugin to check for newer stable releases of Sally. It connects to the public GitHub Releases feed for the 0xeb/sally repository and shows the latest published version.

To check for new versions:

  1. Open the Check For New Versions window:
    Menu:Plugins/Check Version/Check For New Versions...
  2. Click the Check Now button. If a newer release is available, the window shows a direct download link for your platform and a separate link to the GitHub release page.
